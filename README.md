# 📊 Sales Prediction Flask App

A simple Machine Learning web application built using Flask that predicts sales based on input features using a trained regression model.

---

## 🚀 Overview

This project demonstrates an end-to-end Machine Learning workflow, including data preprocessing, model training, and deployment using a Flask web application. Users can input required values and get predicted sales as output.

---

## 🎯 Objective

To build a machine learning model for sales prediction and deploy it as a web application using Flask.

---

## 🛠️ Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS

---

## ⚙️ Working Process

- Data is collected and preprocessed  
- A regression model is trained on the dataset  
- The trained model is saved as a pickle file  
- Flask app loads the model  
- User inputs data through the web interface  
- The model predicts sales and displays the result

## ▶️ How to Run This Project

### 1. Clone the repository
```

git clone [https://github.com/kasayamini/sales-prediction-flask-app.git](https://github.com/kasayamini/sales-prediction-flask-app.git)

```

### 2. Install dependencies
```

pip install -r requirements.txt

```

### 3. Run the application
```

python app.py

```

### 4. Open in browser
```

[http://127.0.0.1:5000/](http://127.0.0.1:5000/)

```
